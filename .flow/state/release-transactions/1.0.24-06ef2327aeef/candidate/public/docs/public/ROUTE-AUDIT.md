# 路线与审计模型

一个公开 `improvementList` 记录表示一条完整配方路线。路线身份包括：

- 基础资源；
- 各阶段开发与改修消耗；
- 消耗品；
- 更新目标和目标等级；
- 二号舰和来源条件。

二号舰专属材料路线会与默认路线分开保存。默认路线记录排除舰船，专属路线记录精确适用舰船；`限定` 选择器不会自动扩展到未来改造。

`dist/data-pipeline/sources/` 保存归一化事实、路线变体、冲突和审计输入。审计结果不会自动覆盖正式数据。

原始网页缓存不进入公开发布。需要复现时，应使用公开来源引用、冻结的 Source Bundle 和当前解析器重新生成。
